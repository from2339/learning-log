# Git/Github学習 
## 2025年3月13日

* ローカルリポジトリを作成  
% git init 

---

* ローカルリポジトリにファイルの変更点をインデックスに登録  
% git add ファイル名  

---

* インデックスに登録したファイルをローカルリポジトリに登録  
% git commit -m "変更点などコメント"  

---

* ローカルリポジトリの変更をリモートリポジトリに反映する  
% git push  

---
